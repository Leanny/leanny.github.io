Quick tutorial to get SplatHeX running

The base of the tutorial was provided by Kiri and modified by me.

Stuff needed

  • A micro SD card that is preferably not from Samsung
  • A savegame manager (e.g. Checkpoint.nro))
  • Fusée Launcher (recomended, for all OSs). Alternative you can use TegraRcmSmash (Download here, GUI version here).
  • Hekate Mod by Jan4V. Download payload.bin and sdfiles.zip.
Kiri prepared the following pacakge which contains most files needed.

Risks

This method involves several risks. If you mess up the RCM boot by using unsave jigs, it might kill your switch. If you do save editing in general, there is always a chance for getting banned. If you only do legal modifications (e.g. add chunks), it will be harder to detect and the probability is lower that you will be banned. Be always aware of the risk this involves.

Step 1:Preparing the SD Card

  • Take the HBL.rar file that you downloaded and extract it. If you downloaded the single packages, extract all of them.
  • Go to HBL\FW 5.0.0, 5.0.1, 5.0.2 and extract the content of the "sdfiles.rar" file and paste it to the root of your SD Card. If you downloaded "sdfiles.rar" as single file, just paste the content of the extracted archive to the root of your SD card.
  • Make a new folder in the root of your sd card and call it "switch".
  • Copy and paste all the file from the savemanager that you downloaded into the switch folder (or in a subfolder if you want it to be more organized)

Step 2: Getting your pc ready

It is recomended to use Fusée Launcher. A tutorial for windows can be found here. Way 1:
  • Go to HBL\Nintendo_Switch_APX_Mode and run "InstallDrivers.exe" as administrator. Hit next then finish.
  • Now open up the "TegraRcmGUI.exe" file in the HBL folder and wait there.
Way 2 (copied from here, prefered when Way 1 does not work):
  • Get your Switch into RCM mode and plug it into your Windows PC. It should show up somewhere (like Device manager) as "APX"
  • Download and run Zadig driver installer from https://zadig.akeo.ie/
  • From the device list, choose APX (if it's not showing up in the list, go to Options menu and check List All Devices)
  • For the driver type, cycle the arrows until you see libusbK (v3.0.7.0) in the text box (IMPORTANT!)
  • Click the big Install Driver button. Device manager should now show "APX" under libusbK USB Devices tree item.

Step 3: Getting your switch into RCM mode

The ultimate list of mods to enter RCM. There are a lot of ways, the most commonly used one is the paperclip method.
Summary of what to do:
  • Turn you switch off.
  • Remove the right joycon.
  • By using a pin, a 3d printed jig or a screwdriver connect the pins (1st and last or last 2) located on the bottom of the right rail. (Just google how to do it properly)
  • While the pin/jig/screwdriver is in place hold down the power button + the VOLUME UP button on the switch until the little switch icon on the program we opened earlier becomes green.

Step 4: Injecting the payload

If you are using Fusée Launcher:
  • Use the following command in the commandline: 
    python fusee-launcher.py payload.bin
If you are using the GUI version of tegrasmash:
  • On the computer click on the little folder icon and choose the file "HBL\FW 5.0.0, 5.0.1, 5.0.2\payload.bin"
  • Click inject payload.
  • On your switch used the power button to choose "launch software" then go down with the volume button to choose "CFW" with the power button again.
If you are using the cmd version of tegrasmash:
  • Drag & Drop payload.bin into tegrasmash.exe
  • On your switch used the power button to choose "launch software" then go down with the volume button to choose "CFW" with the power button again.

Step 5: Dumping the savefile

Important: Do not let the switch go to sleep mode. Otherwise you need to force turn it off due to sleep mode not working properly. To force it to turn off press and hold the power button for around 20 seconds.
  • Your switch should reboot now and when it does open the album.
  • The hombrew menu should boot up, open your savefile manager and dump your savefile.
  • Turn off the switch and remove the sd card.

Step 6: Editing your save

  • Use SplatHeX to edit your savegame.

Step 7: Injecting your save

  • Repeat STEP 4, 5 and 6 but instead of dumping the save, inject it. Make sure to choose the right game/user!
  • Reboot you switch, and start the game.
